# Linux Phone – Site e-commerce

Ce projet est un site web de e-commerce dédié aux téléphones fonctionnant sous Linux (Librem, PinePhone, Volla, Fairphone, etc.). L’objectif était de concevoir une interface moderne, fluide et agréable à
utiliser, en travaillant uniquement avec du HTML, du CSS et du JavaScript.  
Le site comporte plusieurs pages, des animations, une mise en page soignée et une structure claire pour faciliter la navigation.

---

## Fonctionnalités

- Catalogue de téléphones avec images, prix et informations de base  
- Pages dédiées : accueil, boutique, espace compte  
- Animations CSS et transitions sur les éléments interactifs  
- Design responsive (desktop, tablette, mobile)  
- Organisation du code en différents fichiers JS et CSS selon les pages  
- Gestion centralisée des données produits via un fichier JavaScript
